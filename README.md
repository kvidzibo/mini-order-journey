# mini-order-journey
## Getting started
1. Clone the repository:

   ```
   $ git clone https://github.com/kvidzibo/mini-order-journey.git
   ```
2. `cd` to the repository directory:

   ```
   $ cd mini-order-journey
   ```
3. install the dependencies:
   ```
   $ npm install
   ```
4. to run it:
   ```
   $ npm start
   ```
5. to run tests:
   ```
   $ npm test
   ```
